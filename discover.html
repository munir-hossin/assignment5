1. What are the different ways to select an element in the DOM?

I can select elements using:

  • document.getElementById('id')
  • document.getElementsByClassName('class')
  • document.getElementsByTagName('tag')
  • document.querySelector('selector')
  • document.querySelectorAll('selector')

2. What is the difference between innerHTML, innerText, and textContent?

  • innerHTML: Returns or sets HTML content inside an element.
  • innerText: Returns only visible text, ignoring hidden elements.
  • textContent: Returns all text, including hidden elements.

3. What is event delegation in the DOM?

Event delegation is a technique where a parent element listens for events on its child elements instead of adding event listeners to each child individually.

4. What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates up to its ancestors, triggering their event listeners along the way.

5. How do you create, add, and remove elements using JavaScript?

  • Create: document.createElement('tag')
  • Add: parent.appendChild(element) or parent.insertBefore(element, reference)
  • Remove: parent.removeChild(element)
